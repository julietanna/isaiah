Game

Sorting Sorts

This interactive word game asks players to set type to form phrases that are found in Isaiah’s Apprentice Indenture, his first-hand account of the battles of Lexington and Concord, and sayings from early almanacs. Much like early printers, players need to pick the correct pieces of type, or “sorts,” and set them properly in the composing stick so that they will print correctly, all in a set amount of time.
